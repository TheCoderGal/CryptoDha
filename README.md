# Crypto Dha

A simple crypto app using free apis from https://www.coingecko.com to read crypto coin markets data, see the last 7D chart for each coin and more metadata.
Being able to add a coin to portfolio to track.
This uses SwiftUI, Clean architecture, combine.

## ✨ Features

- Markets List of Crypto coins
- Detail View of each coin with an animated chart of last 7D variations
- Portfolio View with ability to add a coin.
- Ability to search the list of coins
